# J&J Entertainment Starter Site

This starter repo is designed for GitHub Pages and a custom domain.

## Suggested repo name
- `jj-entertainment`

## Folder-based service URLs
Because each folder contains its own `index.html`, these URLs work naturally:
- `/photography/`
- `/drone/`

## Publish with GitHub Pages
1. Create a new GitHub repo.
2. Upload these files.
3. In GitHub repo settings, open **Pages**.
4. Set **Deploy from a branch**.
5. Choose **main** and **/(root)**.
6. Add your custom domain.

## GoDaddy DNS
For the root domain (example `jjentertainment.com`), point the apex/root A records to GitHub Pages IPs and set `www` as a CNAME to your GitHub Pages host.
